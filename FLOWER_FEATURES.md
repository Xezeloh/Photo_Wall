# 特殊花朵惊喜功能说明

## 🌸 新增功能概览

这个项目现在包含了三个全新的花朵相关功能模块：

### 1. flowers.js - 花朵飘落动画系统
- **多种花朵类型**：玫瑰、樱花、向日葵、薰衣草
- **自然飘落效果**：模拟真实花朵飘落的物理效果
- **交互生成**：点击和移动鼠标时生成花朵
- **季节适配**：根据不同季节自动调整花朵类型

### 2. flower_surprise.js - 交互式惊喜系统
- **点击惊喜**：点击花朵区域触发各种惊喜内容
- **随机消息**：浪漫情话和温馨提醒
- **照片回忆**：展示特殊纪念照片
- **倒计时功能**：显示重要日子倒计时
- **隐藏彩蛋**：
  - `Ctrl + Shift + F`：花海模式
  - `Ctrl + Alt + H`：秘密花园模式

### 3. seasonal_flowers.js - 季节性主题切换
- **四季主题**：春夏秋冬不同风格
- **节日特典**：情人节、纪念日专属主题
- **一键切换**：右下角花朵按钮快速切换
- **自动检测**：根据日期自动应用合适主题

## 🎯 使用方法

### 基本使用
1. 直接打开网站即可看到花朵飘落效果
2. 点击花朵区域会有惊喜弹出
3. 右下角花朵按钮可以切换季节主题

### 彩蛋功能
- **花海模式**：按 `Ctrl + Shift + F` 开启大量花朵飘落
- **秘密花园**：按 `Ctrl + Alt + H` 在屏幕周围生成大型花朵

### 自定义内容
在 `flower_surprise.js` 中可以修改惊喜内容：
```javascript
this.surprises = [
    { type: 'message', content: '你的自定义消息' },
    // 添加更多惊喜类型...
];
```
